# opensouce
| 명령어  | 기능 요약            | 특징                     |ㅇ
| ---- | ---------------- | ---------------------- |
| top  | 실시간 프로세스 모니터링    | CPU·메모리 사용량을 지속적으로 보여줌 |
| ps   | 현재 실행 중인 프로세스 조회 | 스냅샷 형태로 출력             |
| jobs | 백그라운드 작업 목록 출력   | 쉘 세션 기준 작업 확인          |
| kill | 프로세스 종료 신호 전달    | PID를 통해 종료 가능          |
