# 리눅스 명령어 정리
## 목차
1. [top](#top)
2. [ps](#ps)
3. [jobs](#jobs)
4. [kill](#kill)

---

## 명령어 정리 테이블
| 명령어  | 기능 요약            | 특징                     |
| ---- | ---------------- | ---------------------- |
| top  | 실시간 프로세스 모니터링    | CPU·메모리 사용량을 지속적으로 보여줌 |
| ps   | 현재 실행 중인 프로세스 조회 | 스냅샷 형태로 출력             |
| jobs | 백그라운드 작업 목록 출력   | 쉘 세션 기준 작업 확인          |
| kill | 프로세스 종료 신호 전달    | PID를 통해 종료 가능          |

---

## top
- **설명**

  -현재 시스템의 CPU, 메모리, 프로세스 상태를 실시간으로 모니터링하는 명령어


- **기본 사용법**

  -top

- **주요 단축키**
  
| 단축키 | 설명            |
| --- | ------------- |
| q   | top 종료        |
| k   | 프로세스 kill     |
| h   | 도움말           |
| M   | 메모리 사용량 기준 정렬 |
| P   | CPU 사용량 기준 정렬 |

---

## ps

- **설명**
  
  -현재 실행 중인 프로세스 목록을 출력

- **기본 사용법**
  
  -ps

- **자주 쓰는 옵션**

| 옵션 | 의미                    |
| -- | --------------------- |
| a  | 모든 사용자의 프로세스          |
| u  | 상세 정보                 |
| x  | 터미널 없는 프로세스 포함        |
| -e | 모든 프로세스               |
| -f | 풀 포맷 (UID, PPID 등 표시) |

---

## jobs

- **설명**

  -현재 쉘에서 실행 중인 백그라운드/일시중지 작업 확인

- **기본 사용법**

    -jobs

- **관련 명령어**

| 명령어     | 설명               |
| ------- | ---------------- |
| `&`     | 백그라운드 실행         |
| `fg %1` | 1번 작업을 포그라운드로    |
| `bg %2` | 2번 작업을 백그라운드로 실행 |

---


## kill

- **설명**

  -프로세스에서 신호를 보내 종료시키는 명령어

- **기본 사용법**

  -kill ptd

  -kill -9 ptd #강제 종료

- **자주 쓰는 Signal**

| 번호 | 시그널명    | 설명            |
| -- | ------- | ------------- |
| 9  | SIGKILL | 강제 종료         |
| 15 | SIGTERM | 정상 종료 요청(기본값) |
| 1  | SIGHUP  | 설정 재로딩        |
